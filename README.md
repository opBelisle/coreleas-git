# coreleas-git
The version of Coreleas to clone

This is the first operational version of Corealeas.

Coreleas is an application to help to evaluate
and notate files of computer code.

It's possible to create a scheme with a list of criterions and the point associate (can be negative, positive or 0).  These
criterions are avalaible while evaluation and the scrore is update each times a crierion is choosen.

It's possible to add custum annotation.

The extension of files can be used from now are : .java, .c, .html, .css, .frm,
.js, .py, .cpp, .h, .m, .bas, .txt
